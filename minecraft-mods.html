Minecraft Mod Tutorial

Welcome to the Pixel Academy Modding Tutorial! This page goes through the steps for installing a Forge mod for Minecraft, whether its a popular mod that you heard of on the Internet, or a mod that you yourself made. I created this guide to be followable by children as well as adults, and certain steps have helpful screenshots to make following the guide easier. This guide is for both PC and Mac. If you are having trouble seeing the screenshots, simply click them to open in a new tab.

Please Note: This guide is for installing Forge Mods into Minecraft for version 1.6.4. The steps for different versions of Forge are similar, you simply have to install a different version of Forge, and the rest of the instructions are the same.

Step 1
Please click here to download Forge Mod Loader (or FML)

Step 2

Locate the row labeled 1.6.4-Recommended and on that row, click the link that says Installer. Please see below screenshot for reference.

Step 3
Locate the downloaded item on your computer. By default, you can find this in the downloads folder. Simply double click this downloaded file to start it. It is usually called forge-1.6.4-9.1.1.1.965-installer.jar. Please see screenshot below for what the forge installer program looks like when you double click the above file.

Step 4
Make sure that the "Install Client" radio button is selected (as shown in below screenshot) and click the "OK" button to begin installing Forge. This process may take a couple of minutes to complete. Once the process is complete, Forge has been installed. If there were any errors (For example, it says "You must run Minecraft 1.6.4 at least once") please see the section below labeled Common Issues.

Step 5
Now start the Minecraft launcher by clicking the Minecraft Icon like you normally would when you want to play Minecraft. DO NOT click the "Play" button. there are a few more steps before we are ready to play with mods. See screenshot in step 6 for what the screen should look like.

Step 6
At this point, Forge is installed, but first you must switch to the Forge profile, and afterwards install whatever Mod(s) you would like to play with. To switch to the Forge profile, click the drop down menu labelled Profile: (On the bottom left of the Minecraft Launcher. See Screenshot for reference.) Select the profile labelled Forge in this drop down menu.

NOTE: Whenever you want to play with Mods, simply follow the above step (if not already in the Forge profile). When you want to NOT play with mods, simply repeat the above step, but pick the other profile (that ISN'T Forge).

Step 7
Now we must install the mod(s) that we want to play. This guide does NOT cover downloading mods, and assumes you have the .jar or .zip file needed for the Mod. If using a Mod made by a camper at Camp Pixel, they should have a .zip file that contains their mod. Locate this file (whether its on a flash drive for a camper, or in the downloads folder if you downloaded the mod). Now, Open the Minecraft launcher again. This time, click the button labelled "Edit Profile" in the bottom left corner and a smaller window will pop up. See screenshot for reference on the botton.

Step 8
Now, in the Edit Profile window, click the button labelled "Open Game Dir" (see screenshot for reference) on the bottom right corner. This will open the Game folder that Minecraft uses.

Step 9
Once in this folder, look for a locate a folder called "mods" (all lowercase). If you CANNOT find this folder, simply create a new folder labelled "mods" (again all lowercase). Whether on Windows or Mac, you should be able to simply right click inside the Minecraft folder, and select the "New Folder" option (On windows, you may have to select new, and then select folder in the drop down sub menu)

Step 10
Double click the "mods" folder that was there or that you just made. This is the folder that all mods go into. Simply copy and paste or drag the mod file(s) you located in Step 7 into this folder. Now close the folder, and any other windows that may be open, and start Minecraft normally. Make sure that you are using the Forge profile (See Step 6) and you should be able to start Minecraft and play with the Mod(s) desired)

Common Issues
One of the most common issues, and one that is the most common to occur is the error that happens around Step 4 where, when you try to install forge, an error message pops up that says something to the effect of "You must start Minecraft version 1.6.4 at least once". This error happens alot, especially if you have only recently brought Minecraft, or have recently formatted or wiped your computer, and reinstalled Minecraft. Without going into Much details, you must run the version of Minecraft that Forge is for before installing Forge (So if you want Forge for 1.6.4 you must play regular 1.6.4 at least once.

To fix this error, we must create a new profile. First Open the Minecraft Launcher, then click the button labelled "New Profile" (See screenshot below).

That will open the following Window (See screenshot). Simply give it a name (Probably "Version 1.6.4" if you aren't sure). The name doesn't matter, as long as it's not blank.

Now, select the drop down labelled "Version" (See below screenshot) You want to select version 1.6.4 (Screenshots below) and when finished select the "Save Profile" button in the bottom right corner.

The Menu should look like the following after you have selected the right version:

Now simply click the save profile button in the bottom right corner of the "New Profile" menu. This will close the menu. The Selected profile should automatically change to the profile we created in the above step. If NOT, simply follow step 6 above, but select the profile you just created instead of the Forge profile.Now simply click the "Play" button to start Minecraft. Once Minecraft loads, you can then close it, and follow the directions again.
